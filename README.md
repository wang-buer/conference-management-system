## 1、项目介绍

会议管理系统拥有两种角色

管理员：部门管理、员工管理、会议管理、会议室管理、预订会议、添加员工、注册员工审批等

用户：个人通知中心、预订会议、查看所有会议室等


## 2、项目技术

后端框架： Servlet、mvc模式

前端技术：jsp、css、JavaScript、JQuery

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA、Eclipse都可运行
- tomcat版本：Tomcat 7-10版本均可
- 数据库类型：MySql（5.x和8.x版本都可） 
- maven项目：否
- 硬件环境：Windows 或者 Mac OS
